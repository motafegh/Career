# Deliverable H2 — Implement the bounded parser

Edit `src/aegislab/sshd_parser.py`.

## H2.1 Required imports

Use only:

```python
from __future__ import annotations

import re
from datetime import datetime

from aegislab.event import AuthenticationEvent
```

## H2.2 Required private pattern

Create one compiled regular expression for the bounded grammar. It must use named groups for:

```text
observed_at
process_id
verb
invalid_user
principal
remote_address
remote_port
```

The expression must:

- anchor at start and end;
- require an ISO timestamp as the first token;
- allow an optional `sshd[PID]:` prefix after the timestamp;
- accept only `Accepted` or `Failed`;
- require `publickey`;
- support optional `invalid user` before the principal;
- require `from ADDRESS port PORT ssh2`;
- tolerate trailing key fingerprint or `[preauth]` text without parsing it into unsupported fields.

Do not use a catch-all pattern that turns unrelated lines into events.

## H2.3 Implement `parse_sshd_line`

Required signature:

```python
def parse_sshd_line(
    line: str,
    *,
    environment: str = "namespace",
) -> AuthenticationEvent | None:
```

Required behavior:

1. Preserve the original line without its final newline in `raw_record`.
2. Return `None` for empty lines.
3. Return `None` for lines that do not match the bounded authentication grammar.
4. Parse `observed_at` using `datetime.fromisoformat`.
5. Map `Accepted` to `success`.
6. Map `Failed` to `failure`.
7. Set:

```text
schema_version = "0.1"
environment = supplied argument
source = "openssh-sshd"
service = "sshd"
event_type = "authentication"
auth_method = "publickey"
```

8. Preserve the actual principal.
9. Convert remote port to `int`.
10. Convert optional PID to `int`; otherwise use `None`.
11. Never invent address, port, principal, PID, or timestamp.
12. Add `process_id_missing` to `parse_warnings` when no PID prefix exists.
13. Add `invalid_user_marker_present` when the optional marker appears.
14. Let `AuthenticationEvent` enforce field invariants.

When the wrapper timestamp ends in `Z`, normalize it for `datetime.fromisoformat` without changing the represented instant.

## H2.4 Implement `parse_sshd_file`

Required signature:

```python
def parse_sshd_file(
    path: Path,
    *,
    environment: str = "namespace",
) -> list[AuthenticationEvent]:
```

Required behavior:

1. Open as UTF-8 text.
2. Read in source order.
3. Call `parse_sshd_line` for every line.
4. Append only non-`None` events.
5. Preserve event order.
6. Do not catch file-not-found or permission errors yet; CLI handles them Friday.
7. Do not silently change malformed evidence.

## H2.5 Keep functions small

Target responsibilities:

- regex: recognition and extraction;
- `parse_sshd_line`: mapping one line;
- `parse_sshd_file`: file iteration;
- `AuthenticationEvent`: validation/serialization.

Do not combine CLI printing, file traversal, detection, or quarantine into this module.
