## W2.2 Define `AuthenticationEvent` in `src/aegislab/event.py`

- `dataclasses.dataclass`;
- `datetime.datetime`;
- `json`;
- `typing.Any` only where serialization requires it.

Required public class:

```python
@dataclass(frozen=True, slots=True)
class AuthenticationEvent:
    schema_version: str
    observed_at: datetime
    environment: str
    source: str
    service: str
    event_type: str
    outcome: str
    auth_method: str | None
    principal: str | None
    remote_address: str | None
    remote_port: int | None
    process_id: int | None
    raw_record: str
    parse_warnings: tuple[str, ...] = ()
```

Required invariants in `__post_init__`:

1. `schema_version` is not empty.
2. `observed_at` is timezone-aware; naive datetimes are rejected.
3. `environment` is not empty.
4. `source` is not empty.
5. `service` must equal `sshd` in Week 1.
6. `event_type` must equal `authentication` in Week 1.
7. `outcome` must be exactly `success` or `failure`.
8. `remote_port`, when present, is between 1 and 65535.
9. `process_id`, when present, is greater than 0.
10. `raw_record` is not empty after whitespace stripping.
11. `parse_warnings` contains only nonempty strings.

Do not silently coerce invalid values. Raise `ValueError` with a specific field-oriented message.

Required methods:

```python
def to_dict(self) -> dict[str, Any]:
    ...

def to_json(self) -> str:
    ...
```

Serialization contract:

- `observed_at` becomes ISO 8601 text;
- `parse_warnings` becomes a JSON array;
- `None` remains JSON `null`;
- keys remain stable and match field order;
- output contains no derived verdict or detection.
