## M4.2 Execute the bounded loop

```bash
FAILURE_CLIENT_LOG="$EVIDENCE_TMP/client-repeated-failure.log"
: > "$FAILURE_CLIENT_LOG"

STARTED_AT=$(date --iso-8601=seconds)
printf 'STARTED_AT=%s\n' "$STARTED_AT" | tee "$EVIDENCE_TMP/failure-start.txt"

for attempt in 1 2 3 4 5; do
  printf '\n=== ATTEMPT %s START %s ===\n' \
    "$attempt" \
    "$(date --iso-8601=seconds)" |
    tee -a "$FAILURE_CLIENT_LOG"

  sudo ip netns exec aegis-test \
    sudo -u motafeq \
    ssh \
    -vv \
    -T \
    -i "$LAB_ROOT/client/unauthorized_id_ed25519" \
    -o IdentitiesOnly=yes \
    -o BatchMode=yes \
    -o StrictHostKeyChecking=yes \
    -o UserKnownHostsFile="$LAB_ROOT/client/known_hosts" \
    -o ConnectTimeout=5 \
    motafeq@10.10.0.2 true \
    2>&1 |
    tee -a "$FAILURE_CLIENT_LOG"

  status=${PIPESTATUS[0]}
  printf 'ATTEMPT=%s EXIT_STATUS=%s END=%s\n' \
    "$attempt" \
    "$status" \
    "$(date --iso-8601=seconds)" |
    tee -a "$FAILURE_CLIENT_LOG"

  sleep 2
done

ENDED_AT=$(date --iso-8601=seconds)
printf 'ENDED_AT=%s\n' "$ENDED_AT" | tee "$EVIDENCE_TMP/failure-end.txt"
```

Expected: each client command fails authentication with a nonzero status. TCP and server-host verification may still succeed.
