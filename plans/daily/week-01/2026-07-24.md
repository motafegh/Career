# Deliverable F1 — Record the CLI contract before implementation

In the Friday AegisLab worklog section `CLI contract`, record:

## Command surface

```text
python -m aegislab.cli parse PATH [--environment ENVIRONMENT]
aegislab parse PATH [--environment ENVIRONMENT]
```

## Standard streams

- standard output contains JSON Lines only;
- one normalized event per line;
- no headings, progress text, debug text, or explanations on standard output;
- diagnostics and error descriptions go to standard error.

## Exit codes

```text
0 = file read successfully and at least one authentication event emitted
1 = file read successfully but no supported authentication events found
2 = argparse usage error
3 = file I/O error
4 = event parsing or validation error
```

## JSON contract

- every stdout line is one valid JSON object;
- keys match `AuthenticationEvent.to_dict()`;
- output order matches fixture record order;
- `null` represents missing optional data;
- warnings are arrays;
- no detection, alert, score, or verdict is added.

## Required fixture results

```text
success.log          → exit 0 → at least 1 JSON line → every outcome success
unauthorized_key.log → exit 0 → exactly 5 JSON lines → every outcome failure
```

Do not implement until this contract is written.


Use required subparsers so running without a subcommand is a usage error.

## F2.3 JSON emission helper

Create:

```python
def _emit_jsonl(events: Sequence[AuthenticationEvent]) -> None:
```

When type imports create a circular dependency, import `AuthenticationEvent` directly from `event.py`; do not use `Any` to hide the problem.

Required emission:

```python
print(
    json.dumps(
        event.to_dict(),
        ensure_ascii=False,
        separators=(",", ":"),
    )
)
```

One `print` call per event.

## F2.4 Implement `main`

Required signature:

```python
def main(argv: Sequence[str] | None = None) -> int:
```

Required behavior:

1. Parse arguments.
2. On `parse`, call `parse_sshd_file(path, environment=...)`.
3. Catch `OSError`:
   - print `error: unable to read <path>: <message>` to `sys.stderr`;
   - return 3.
4. Catch `ValueError` from parsing/validation:
   - print `error: invalid authentication evidence in <path>: <message>` to `sys.stderr`;
   - return 4.
5. When no events are returned:
   - print `warning: no supported authentication events found in <path>` to `sys.stderr`;
   - return 1.
6. Otherwise emit JSONL and return 0.
7. Do not call `sys.exit()` inside `main`.
8. Keep:

```python
if __name__ == "__main__":
    raise SystemExit(main())
```
